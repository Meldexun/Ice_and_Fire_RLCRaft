    作为自然元素的征象，爬行巨兽————
无论冰火————一直在人类社会中作为传奇
而广为人知。但有传闻说，还有一种巨兽
与它们相似，却只栖息于最奇异的环境中。
它们，便是电龙。




<image><drawings_lightningdragon.png 0 80 0 0 194 93 1>






与它们掌握冰或火力量的表亲不同
电龙 是夜行性动物，出没于丛林漆黑夜空中。
电龙居住在由霆霹方块构成的巢窠或地下
洞穴中。就像冰龙和火龙非常喜欢某些宝石一样，
电龙也有自己最喜欢的宝石，那就是紫水晶和钻石。

除了在丛林中发现之外，
还有报道称在丛林之外也发现了闪电龙：
在高山、深峡谷、干燥荒地，
甚至在雷暴期间。不过
这些报道尚未得到证实。
